Core Sero Engine Handoff Bundle

Goal: give another AI enough context to understand how the 1:1 engine works.

Included:
- Engine pipeline code under src/
- Meeting type registry + all 5 type definitions
- Core prompts used by each pipeline stage
- Question bank index + question YAML files (q_*.yaml)
- Runtime entry points (cli.js, smoke-test.js)
- Model config and project docs for orientation

Excluded on purpose:
- Frontend app files
- Logs/archives
- node_modules
- unrelated workspace/agent config
