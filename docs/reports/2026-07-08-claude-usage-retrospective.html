Your new toolbox — one word each

Say checkpoint
Mid-work: saves everything (commit + STATUS + chat-log). In a fresh session: brings the whole picture back — done, awaiting QA, your move.
checkpoint skill
Say green light / tested, good
Runs the full phase close-out: plan, STATUS, board, badges, log, changelog, commit — nothing skipped, nothing drifts.
phase-close skill
Say night test
The full overnight QA sweep — free checks first, paid runs capped and tallied, report waiting in the morning.
night-test skill
Say commit my work
Path-scoped commits of this session's files only. If foreign changes are mixed in, you get an A/B/C choice — never a silent sweep.
safe-commit skill
Say review this run
Full run review — and it now builds the Report/Role/Meeting-type context block itself. No more pasting it.
reviewrun skill
Say darren method / split into phases
Plan → phases → one at a time, you green-light each. Templates now include GREEN-LIT/Built sections and a findings template for audits.
darren-method skill
Say start the app
Dev servers via the preview tools — no more 26 throwaway "localhost" sessions.
