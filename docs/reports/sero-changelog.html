The pre-go-live "manager-ready" track — turning the internal tool into something a real manager can be handed. Foundation phases are done; this is the make-it-shippable pass. Most items below are built and awaiting Carl's QA walk.

The core

The heart of Sero: prepare a great 1:1, run it live, and get an honest wrap-up — in your team's own language.

The engine and the manager-facing run that everything else is built around.

  • One-page run. The whole conversation on a single page that grows as you go.
  • Your team's words. Teach Sero the vocabulary your roles actually use (job lexicons).
  • Five kinds of 1:1 — bi-weekly, performance, growth, "something feels off," and onboarding — each with its own natural shape.
  • Live scoring on four things — wellbeing, engagement, clarity, growth — earned only from what's actually said.
  • The pipeline: focus areas → prep brief → question bank → live per-turn planner → final briefing (+ optional phrase harvest).
  • Content: a 4,400+-question library, 31 cached role profiles, neutral-seed axes, dedup + question-integrity gates.
  • QA cockpit: run library, side-by-side Compare, 8-dimension verdicts, and an offline regression check.
