# 📋 The board — moved

This hand-maintained kanban kept drifting behind the real trackers (it listed closed
tracks as "your move" for days). It's **retired** to stop the "which board is real?"
confusion. Use the sources that stay true:

- **Live plan board** — the **Tasks** screen in the admin app, auto-built from the
  `docs/plans/{doing,done,future}/` folders (page-heartbeat track). It can't drift.
- **[STATUS.md](../STATUS.md)** — what's happening right now (the tactical tracker).
- **[SERO_BOARD.md](../SERO_BOARD.md)** — the strategic map.

At a glance, plan state lives in the folders themselves:
`docs/plans/doing/` = actively in-flight · `docs/plans/done/` = closed ·
`docs/plans/future/` = parked. For the live picture, read [STATUS.md](../STATUS.md).
