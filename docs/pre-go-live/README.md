# Pre-go-live

The track that turns Sero from "runs one 1:1" into "a tool a manager keeps using" — their people,
their past 1:1s, ratings — plus a window for Carl on the whole alpha.

1. Open **[OVERVIEW.md](OVERVIEW.md)** in your AI agent — that's the playbook that drives the work.
2. Say **"what next?"** — it reads the log, breaks down the current phase, then does one step at a time.
3. After each step, do the quick check it gives you; when a phase is done, it asks you to approve before moving on.
4. Keep saying **"what next?"** to continue. Progress and decisions are tracked in **[PROGRESS.md](PROGRESS.md)**.
5. If anything's unclear or breaks, just tell the agent in plain words — don't force it.

Same rhythm as [prototype-to-production](../archives/prototype-to-production/OVERVIEW.md): **one phase at a time,
you test and green-light each before the next.**
